

1、安装nodejs：自己下载，官网

2、检测：地址栏输入cmd回车，指令窗口弹出来，输入指令：node -v(出现版本号，安装成功)

3.我们安装node的目的，想用npm模块，专门用于下载资源，输入指令：npm -v(出现版本号，安装成功)

4、利用npm安装自己需要的模块：需要gulp模块(项目自动化构建：压缩文件、归并文件、编译文件、刷新文件、监控文件)

安装gulp：全局(安装在node目录下，在各个盘都能使用)、本地(安装在开发目录下)

-方法一：一个一个安装

	全局安装：npm install 模块名(包名) -g
	
	全局安装gulp：npm install gulp -g
	
	本地安装：npm install gulp --save -dev
	
	安装插件：安装压缩css插件：npm install gulp-cssmin --save -dev 

	--save(你安装了gulp,就把这个下载过的模块，记录到package.json,方便后期安装使用)
	
-方法二：一次性安装：利用package.json文件，放在根目录，就可以一次性安装（只要写好安装指令，电脑就会自动识别你的package.json的所有需要的模块，全部帮你下载安装），因为模块之间存在依赖关系，所有安装gulp的时候，会发现很多其他模块一起被安装

	指令：npm install(package.json有什么，就安装什么)
	
5.用gulp进行自动化项目构建：

了解gulp五个方法：

task() 任务
src() 源文件路径
pipe() 管道：处理文件
dest() 目标路径
watch() 监听：自动监听和更新文件

写好gulpfile.js文件，执行里面的任务完成项目构建；
