

服务器：硬件

1.服务器是一台特别的电脑：硬件要好

分类：

1.自己买一台：大企业

2.租用：租一个服务器，买了放在电信托管；或者直接租服务器；中小型企业

3.租空间，云服务器：分布式、集群，阿里云、腾讯云、百度云、谷歌云、华为云

服务：

服务器：软件

ftp服务：上传、下载文件，文件共享

svn服务：文件共享/版本管理,集中式管理

	-图书管理系统最终版
	-图书管理系统最最终版
	-图书管理系统终极完美版
	-图书管理系统打死不改版
	-图书管理系统升级版

git服务：文件共享/版本管理，分布式管理

web服务：网站服务，不同的后端语言需要不同的环境

git工具：https://blog.csdn.net/weixin_37887248/article/details/81011364


如果用vs code 的git工具进行代码管理；

1.安装git工具；

2.在你所以的项目最外层目录下初始化仓库；git init

3.在github创建一个空的仓库；

写指令把本地仓库和远程仓库连接在一起：git remote add origin 仓库地址

git pull

4.用vs code打开你的项目(资源管理器可以看到你的项目目录)

5.开发项目：

-如果开发好一个版本，就可以提交了：到git工具(源代码管理)进行提交和代码托管

	- 写日志
	- 打钩
	-左下角点击：同步(刷新按钮) 自带push和pull

教程：https://www.cnblogs.com/richard1015/p/8336429.html
